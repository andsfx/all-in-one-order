# Panduan Setup Toko

Panduan ini untuk pemilik bisnis yang baru pertama kali menggunakan sistem. Ikuti langkah-langkah berikut untuk membuka toko online Anda.

## Langkah 1: Login

1. Buka link admin yang diberikan (contoh: `https://namatoko.vercel.app/login`)
2. Masukkan email dan password yang sudah disiapkan
3. Klik "Masuk"

## Langkah 2: Setup Wizard

Setelah login pertama kali, Anda akan diarahkan ke Setup Wizard:

1. **Pilih Jenis Bisnis** — Pilih yang paling sesuai (Kedai Minuman, Restoran, Toko Fisik, atau Produk Digital)
2. **Nama Toko** — Isi nama toko dan nomor WhatsApp Anda
3. **Upload Logo** — Upload logo toko (opsional, bisa ditambah nanti)
4. **Pembayaran** — Upload gambar QRIS Anda untuk terima pembayaran
5. **Tipe Pemenuhan** — Pilih cara pengiriman (dine-in, takeaway, delivery)
6. **Jam Operasional** — Atur jam buka dan tutup
7. **Cabang Pertama** — Isi nama dan alamat cabang
8. **Selesai** — Klik "Buka Toko" untuk mulai!

## Langkah 3: Edit Produk Sample

Setelah wizard selesai, sistem akan membuat produk sample sesuai jenis bisnis Anda. Produk ini ditandai badge "SAMPLE".

**Cara edit produk sample:**
1. Buka menu **Kelola Menu** di sidebar
2. Klik ikon pensil pada produk yang ingin diedit
3. Ganti nama, harga, dan deskripsi sesuai menu asli Anda
4. Upload foto produk asli
5. Klik "Simpan"

**Cara hapus semua sample sekaligus:**
1. Buka **Pengaturan**
2. Scroll ke bagian "Kelola Data Sample"
3. Klik "Hapus Semua Data Sample" atau "Jadikan Milik Saya"

## Langkah 4: Tambah Varian

Jika produk Anda punya pilihan (ukuran, rasa, topping):
1. Buka **Kelola Menu**
2. Klik ikon varian pada produk
3. Tambah varian dengan nama dan harga

## Langkah 5: Test Order

Sebelum bagikan ke pelanggan:
1. Buka link toko (tanpa /admin) di browser atau HP
2. Coba pesan 1 produk
3. Cek apakah pesanan muncul di halaman admin
4. Pastikan semua berjalan lancar

## Langkah 6: Bagikan Link Toko

Toko Anda sudah siap! Bagikan link ke pelanggan via:
- WhatsApp
- Instagram bio
- Cetak QR code

## FAQ

### Bagaimana ganti nama toko?
Buka **Pengaturan** → bagian Branding → ganti nama → Simpan.

### Bagaimana ganti warna tema?
Buka **Pengaturan** → bagian Branding → pilih warna → Simpan.

### Bagaimana tambah kategori baru?
Buka **Kelola Menu** → klik "Tambah Kategori".

### Bagaimana nonaktifkan produk sementara?
Buka **Kelola Menu** → klik toggle mata pada produk.

### Bagaimana tutup toko sementara?
Di halaman admin utama, klik toggle "Buka/Tutup Toko".
